## Описание
Адаптивный клон главнойстраницы YouTube, созданный с использованием HTML, CSS, Flexbox и CSS Grid.". Лабораторная подразделяется на три части, в которых входит изучение: грид разметки, работы с HTML, написание кода СSS для визуального оформления страницы. Страница содержит основное меню оригинального сайта YouTube, строку поиска, а также примеры аидео-материалов.
